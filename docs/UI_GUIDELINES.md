# UI Guidelines

Style:
- Retro-modern
- Neon synthwave
- Dark mode focused

Colors:
- Black background
- Neon pink accents
- Deep red highlights

Design Principles:
- Minimal clutter
- Immersive playback
- Smooth transitions
- Strong glow effects
- Rounded cards
- Large album art
- Persistent mini-player

Animations:
- Vinyl rotation
- Soft pulsing glow
- Animated seekbar
- Smooth navigation transitions
